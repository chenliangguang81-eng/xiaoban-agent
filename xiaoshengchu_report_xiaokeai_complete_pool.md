## 二、 一派+二派联动填报策略

一派是“提前批”，录取概率受名额限制，但一旦录取即锁定，无需参与二派摇号。二派是全区派位，池内6所学校质量均在B类及以上，**没有坑校**。

### 💡 核心逻辑：一派大胆冲高，二派稳妥排序

因为二派的底线非常高（最差也是育英中学/玉渊潭中学），所以**一派绝对不要填报 B 类及以下的学校**，否则等于主动放弃了冲刺顶级名校的机会，且浪费了二派优质的保底池。

### 方案 A：激进冲顶（目标直指顶级名校）

适合小可爱成绩优异，且家庭愿意承担一定摇号风险。

**【一派志愿】（只填 A 类及以上）**
1. **十一晋元**（如果有"密电"或特殊渠道，否则不填）
2. **五十七中** 或 **人大附中翠微学校**（二选一，根据对"严格管理"还是"人大附光环"的偏好决定）
3. **101中学双榆树校区** 或 **交大附中**

**【二派志愿】（A → A → A- → B+ → B → B）**
1. **五十七中**
2. **人大附中翠微学校**
3. **理工附中（东校区）**
4. **育鸿学校**
5. **玉渊潭中学**
6. **育英中学**

### 方案 B：稳妥锁定（不想参与二派摇号的煎熬）

适合小可爱成绩中上，家庭希望尽早确定学校，避免二派的不确定性。

**【一派志愿】（A- 类打底）**
1. **五十七中** 或 **人大附中翠微学校**（冲刺）
2. **交大附中分校** 或 **首师大附中一分校**（稳妥）
3. **十一实验中学**（稳妥）

*注：即使是稳妥方案，一派也不建议填报育鸿学校、玉渊潭中学或育英中学，因为这些学校在二派中作为保底已经足够安全。*

## 三、 核心行动项与关注点

1. **五十七中 vs 人翠的抉择**：
   - 如果小可爱**抗压能力强**，能适应较大作业量和严格管理，首选 **五十七中**。
   - 如果小可爱**更看重综合素质**和人大附中的教学理念，首选 **人翠**。

2. **玉渊潭中学的战略价值**：
   - 虽然被排在二派的第五志愿，但玉渊潭中学的 **1+3 项目（初二直升高中）** 对中游学生极具性价比。如果小可爱在初一、初二成绩处于中上游，通过玉渊潭的 1+3 提前锁定高中，能完美规避中考的惨烈竞争。

3. **心态建设**：
   - 七一小学的派位池堪称“神仙池”，**无论摇中哪一所，都是区重点以上的水平**。请 Lion 和小可爱完全放下焦虑，把精力集中在**小初衔接的数学和英语**上，为初一的分层考试（分班考）做好准备。
