## 一、政策核心要点（2026年无重大变化）

| 要点 | 内容 |
|---|---|
| **本质** | 初二学生提前锁定优质高中，读1年初三+3年高中，四年一贯制 |
| **中考** | 不用参加传统中考，只需通过学业水平考试（合格即可升学） |
| **报名资格** | 具有本市普高升学资格 + 同一学校连续2年初中学籍的初二学生 |
| **报名数量** | 最多报1所市级实验校 + 1所区级实验校 |
| **录取顺序** | 先录市级，再录区级 |
| **退出机制** | 录取后可随时退出，回原校参加中考，不影响后续升学 |
| **选拔方式** | 严禁考文化课，按综合素质评价1:3比例确定面试名单，再面试录取 |

## 六、综合素质评价（综素）是核心竞争力

1+3录取**严禁考文化课**，完全依据综合素质评价筛选面试名单。综素档案应包含：
- 社会实践记录
- 获奖证书（科技类、人文类、体育类均可）
- 研究性学习报告
- 班级职务、志愿服务记录

**结论**：从现在（小学六年级）开始，就应有意识地积累综素材料，而不是等到初二再临时抱佛脚。
