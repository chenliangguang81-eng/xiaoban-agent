# 张雪峰学习方法论核心语料

## 1. 学习是反人性的
学习本身就是一件反人性的事情，不要指望孩子能"快乐学习"。
真正的快乐来自于解出一道难题后的成就感，来自于成绩提升后的自信。
**执行力训练法**：
- 不要考验意志力，要改变环境。手机放客厅，书桌上只放书和笔。
- 番茄工作法：专注25分钟，休息5分钟。
- 睡前10分钟：像放电影一样在脑子里过一遍今天学了什么，想不起来的明天一早看。

## 2. 各科提分策略
### 语文
- **作文**：不要背范文，要背"素材"。一个好素材（比如某个历史人物的冷门故事）可以套用在"坚持"、"创新"、"爱国"等多个主题上。
- **结构**：凤头（开头要惊艳）、猪肚（中间内容要充实）、豹尾（结尾要升华扣题）。
- **阅读理解**：带着问题去读文章，先看题目问什么，再回原文找答案。

### 数学
- **刷题误区**：不要盲目刷数量，要刷"类型"。做对10道同类型的题，不如把1道题的3种解法搞懂。
- **错题本**：错题本不是用来抄题的，是用来分析"我为什么错"的。是因为概念不清？计算失误？还是没看懂题意？
- **几何**：拿到几何题，第一步永远是画图，把已知条件标在图上。

### 英语
- **单词**：不要孤立地背单词，要放在句子里背。
- **听力**：每天坚持听15分钟，磨耳朵。
- **作文**：准备几个高级句型模板，考试时往里套词。

## 3. 考试心法
- **心态**：考试不是表演，是信息提取。紧张是正常的，深呼吸，先做自己最有把握的题。
- **时间分配**：先易后难。遇到卡壳的题，超过3分钟没思路就果断跳过，不要死磕。
- **检查**：数学考试最后5分钟，不要去想那道做不出的压轴题，去检查前面的计算题，确保该拿的分拿到。

## 4. 读书方法论
- **文科阅读**：带着批判性思维去读。作者的观点是什么？他的论据能支撑他的观点吗？我同意吗？
- **理科阅读**：先看目录和每章小结，了解整体框架，再深入细节。公式推导一定要自己动手推一遍。
- **经典书单**：小学阶段多读经典文学（如《窗边的小豆豆》、《夏洛的网》），培养阅读兴趣和语感。初中开始接触科幻（如《三体》）和历史。
